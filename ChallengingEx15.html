Challenging Exercise 15: JavaScript Class Implementation

Description

Exercise: Create a modern JavaScript class named `Vehicle` with a constructor to set properties and a method to describe itself, demonstrating object-oriented concepts.

Solution Code Snippet


class Vehicle {
    constructor(make, model, year) {
        this.make = make;
        this.model = model;
        this.year = year;
    }

    describe() {
        return `This is a ${this.year} ${this.make} ${this.model}.`;
    }
}

const myCar = new Vehicle("Toyota", "Prius", 2020);
const description = myCar.describe();
