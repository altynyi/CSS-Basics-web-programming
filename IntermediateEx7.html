Intermediate Exercise 7: Flexbox Navigation Layout

Description

Exercise: Create a responsive, evenly spaced navigation bar using CSS Flexbox properties like `display: flex;` and `justify-content: space-between;`.

Solution Code Snippet


/* In css/styles.css: */
.ex7-nav {
    display: flex;
    justify-content: space-between; /* Spreads items out */
    align-items: center;
}
