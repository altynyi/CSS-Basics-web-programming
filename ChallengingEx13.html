Challenging Exercise 13: Fetch API Usage (Simulated)

Description

Exercise: Write an asynchronous function using `async`/`await` and the `fetch()` API to retrieve and display data from a simulated API endpoint (using `jsonplaceholder.typicode.com`).

Solution Code Snippet


async function fetchData() {
    try {
        const response = await fetch('https://jsonplaceholder.typicode.com/posts/1');
        const data = await response.json();
        
        document.getElementById('ex13-title').textContent = data.title;
        document.getElementById('ex13-body').textContent = data.body;
    } catch (error) {
        console.error('Fetch error:', error);
    }
}

fetchData();
