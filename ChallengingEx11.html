Challenging Exercise 11: Debounce Function Implementation

Description

Exercise: Implement a `debounce` function to limit the rate at which a function (like a search handler) is called. The function should only execute after a specified delay following the last event call.

Solution Code Snippet


function debounce(func, delay) {
    let timeoutId;
    return function(...args) {
        clearTimeout(timeoutId); // Clear previous timeout
        timeoutId = setTimeout(() => {
            func.apply(this, args);
        }, delay);
    };
}

// Example usage:
const handleExpensiveInput = debounce((value) => {
    console.log("Processing search for:", value);
    // ... expensive API call here
}, 500);
