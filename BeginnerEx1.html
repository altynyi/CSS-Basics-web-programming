Beginner Exercise 1: Semantic Structure

Description

Exercise: Create the basic semantic shell of an HTML5 document, including core structural elements like the header, main content, and footer.

Solution Code Snippet


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    <header>...</header>
    <main>
        <section>...</section>
    </main>
    <footer>...</footer>
</body>
</html>
