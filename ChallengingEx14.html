Challenging Exercise 14: ARIA Role for Live Alert

Description

Exercise: Use the WAI-ARIA `role="alert"` attribute on a `div` element to ensure important, dynamic status messages are immediately announced to screen readers without requiring the user to shift focus.

Solution Code Snippet


<div role="alert" id="status-message">
    Warning: Form data is incomplete.
</div>

/* JavaScript updates the content to trigger the announcement. */

Live Result Demonstration

System Status: Waiting for user action...

When the warning is triggered, a screen reader would immediately announce the updated text due to `role="alert"`.
