Intermediate Exercise 8: Form Validation

Description

Exercise: Build an input field that requires a minimum length of 8 characters using declarative HTML validation attributes.

Solution Code Snippet


<form onsubmit="return false;">
    <label for="pass">Password (Min 8 Chars):</label>
    <input type="password" 
           id="pass" 
           required 
           minlength="8">
    <button type="submit">Submit</button>
</form>

Live Result Demonstration



Try submitting with fewer than 8 characters to see the browser validation message.
