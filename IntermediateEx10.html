Intermediate Exercise 10: CSS Transform & Transition

Description

Exercise: Apply a slight scale-up and rotation effect when a user hovers over an element, using `transition` for a smooth visual experience.

Solution Code Snippet


.ex10-box {
    transition: transform 0.3s ease-in-out; /* Enables smooth animation */
}
.ex10-box:hover {
    transform: scale(1.1) rotate(2deg); /* Scales up and rotates slightly */
}
