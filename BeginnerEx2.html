Beginner Exercise 2: Element Styling

Description

Exercise: Change the background color of a specific `div` element using its class selector in external CSS.

Solution Code Snippet


/* In styles.css or linked style block: */
.ex2-div {
    background-color: #f0f8ff; /* Demonstrates styling via class selector */
    padding: 15px;
    border-radius: 4px;
}
