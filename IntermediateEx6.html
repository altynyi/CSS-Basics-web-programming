Intermediate Exercise 6: Factorial Recursion

Description

Exercise: Calculate the factorial of a number ($n!$) using a recursive JavaScript function. We will calculate $5!$.

Solution Code Snippet


function factorial(n) {
    // Base case: 0! is 1
    if (n === 0 || n === 1) {
        return 1;
    }
    // Recursive step: n * factorial(n - 1)
    return n * factorial(n - 1);
}

// Calculate 5!
const result = factorial(5); // Should be 120
