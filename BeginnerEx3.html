Beginner Exercise 3: JS Addition Function

Description

Exercise: Write a basic JavaScript function to add two numbers and display the result.

Solution Code Snippet


function add(a, b) {
    return a + b;
}

const result = add(5, 12); // result is 17

document.getElementById('result').textContent = result;
