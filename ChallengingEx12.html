Challenging Exercise 12: Responsive CSS Grid Layout

Description

Exercise: Use CSS Grid to create a responsive dashboard that displays four items in a $2\times2$ grid on desktop, a single row of 4 columns on tablet, and a single column on mobile. (The global CSS only shows the mobile to 3-column switch for simplicity).

Solution Code Snippet


/* In css/styles.css: */
.ex12-grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* Default 3 columns (desktop) */
    gap: 15px;
}

@media (max-width: 768px) {
    .ex12-grid {
        grid-template-columns: 1fr; /* Switch to 1 column on mobile */
    }
}
